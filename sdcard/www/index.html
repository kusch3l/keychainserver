KeyChainServer

Dieser Server ist quasi ein komplett eingenständiges Internet und nicht mit dem WorldWideWeb verbunden sondern allein auf diesem kleinen Schlüsselanhänger. In diesem befindet sich ein ESP31 Mikrocontroller mit eine Knopfzelle und einer MicroSD-Karte. Wenn du genauer verstehen möchtest, wie das ganze funktioniert, rede mitkusch3l, die kann dir das besser erklären.
